# Ecommerce Sales Analysis Project

I built this project to show my data analysis skills.  
I use real sales data from Brazil.  
I analyze order trends and customer locations.  

## Project Goal  
I find monthly sales changes.  
I understand customer distribution.  
This helps companies improve inventory and marketing.  

## Data Source  
I download the dataset from Kaggle.  
The name is Brazilian E-Commerce Public Dataset by Olist.  
It contains orders, items and customers from 2016 to 2018.  

## Tech Stack  
I use Tableau Public for visualization.  
I use COUNT([order_id]) for sales count.  
I use COUNT([customer_id]) for customer count.  

## Steps I Followed  
I connect three CSV files in Tableau.  
I join tables on order_id and customer_id.  
I create a line chart for monthly sales.  
I create a bar chart for customers by state.  
I combine charts into one dashboard.
